This guide will walk you through creating your first carousel, from basic setup to adding advanced features.

## Prerequisites

Before you begin, make sure you have:
- A React project (version 19.0.0 or higher)
- Node.js 18 or higher
- npm, yarn, or pnpm installed

<Note>
If you haven't installed AnyCarousel yet, check out the [Installation](/installation) guide first.
</Note>

## Build Your First Carousel

<Steps>
  <Step title="Import the Carousel component">
    Start by importing the Carousel component into your React file:

    ```tsx
    import { Carousel } from "react-any-carousel";
    ```
  </Step>

  <Step title="Create a basic carousel">
    Add the Carousel component with some simple content:

    ```tsx
    function App() {
      return (
        <Carousel>
          <div className="card">Slide 1</div>
          <div className="card">Slide 2</div>
          <div className="card">Slide 3</div>
          <div className="card">Slide 4</div>
        </Carousel>
      );
    }
    ```

    That's it! You now have a working carousel with navigation buttons and swipe support.
  </Step>

  <Step title="Add some styling">
    Style your carousel items so they're visible:

    ```css
    .card {
      min-width: 300px;
      height: 200px;
      background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
      border-radius: 12px;
      display: flex;
      align-items: center;
      justify-content: center;
      color: white;
      font-size: 24px;
      font-weight: bold;
      margin: 0 10px;
    }
    ```

    <Tip>
    The `min-width` property ensures your items maintain their size, while `margin` adds spacing between items.
    </Tip>
  </Step>

  <Step title="Configure snap scrolling">
    Add the `scrollSnapType` prop to control how items snap into place:

    ```tsx
    <Carousel scrollSnapType="center">
      <div className="card">Slide 1</div>
      <div className="card">Slide 2</div>
      <div className="card">Slide 3</div>
      <div className="card">Slide 4</div>
    </Carousel>
    ```

    Available snap options:
    - `"start"` - Items snap to the start of the container
    - `"center"` - Items snap to the center (great for galleries)
    - `"end"` - Items snap to the end
    - `"none"` - Free scrolling without snapping
  </Step>

  <Step title="Control scroll distance">
    Use `scrollOffset` to control how far the carousel scrolls with each click:

    ```tsx
    <Carousel 
      scrollSnapType="center" 
      scrollOffset={320}
    >
      <div className="card">Slide 1</div>
      <div className="card">Slide 2</div>
      <div className="card">Slide 3</div>
      <div className="card">Slide 4</div>
    </Carousel>
    ```

    <Note>
    Set `scrollOffset` to your card width plus margin for single-item scrolling, or to the container width for full-page scrolling.
    </Note>
  </Step>

  <Step title="Customize animation">
    Adjust the scroll `duration` and add custom `scrollEasing` for smooth animations:

    ```tsx
    <Carousel 
      scrollSnapType="center" 
      scrollOffset={320}
      duration={800}
      scrollEasing="cubic-bezier(0.25, 1, 0.5, 1)"
    >
      <div className="card">Slide 1</div>
      <div className="card">Slide 2</div>
      <div className="card">Slide 3</div>
      <div className="card">Slide 4</div>
    </Carousel>
    ```

    The `duration` is in milliseconds (range: 100-5000ms, default: 600ms).
  </Step>
</Steps>

## Complete Example

Here's a complete, production-ready carousel with all features:

```tsx App.tsx
import { Carousel } from "react-any-carousel";
import "./styles.css";

function App() {
  return (
    <div className="app">
      <h1>My Carousel</h1>
      <Carousel
        scrollSnapType="center"
        scrollOffset={320}
        duration={800}
        scrollEasing="cubic-bezier(0.25, 1, 0.5, 1)"
        theme="light"
      >
        <div className="card">
          <img src="/images/slide1.jpg" alt="Slide 1" />
          <h3>Beautiful Slide 1</h3>
        </div>
        <div className="card">
          <img src="/images/slide2.jpg" alt="Slide 2" />
          <h3>Amazing Slide 2</h3>
        </div>
        <div className="card">
          <img src="/images/slide3.jpg" alt="Slide 3" />
          <h3>Stunning Slide 3</h3>
        </div>
        <div className="card">
          <img src="/images/slide4.jpg" alt="Slide 4" />
          <h3>Gorgeous Slide 4</h3>
        </div>
      </Carousel>
    </div>
  );
}

export default App;
```

```css styles.css
.app {
  max-width: 1200px;
  margin: 0 auto;
  padding: 40px 20px;
}

h1 {
  text-align: center;
  margin-bottom: 40px;
  color: #333;
}

.card {
  min-width: 300px;
  height: 400px;
  background: white;
  border-radius: 12px;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
  margin: 0 10px;
  overflow: hidden;
  display: flex;
  flex-direction: column;
}

.card img {
  width: 100%;
  height: 300px;
  object-fit: cover;
}

.card h3 {
  padding: 20px;
  margin: 0;
  text-align: center;
  color: #333;
}
```
